Introduction

What is Markdown?

Markdown is a lightweight markup language designed for easy formatting of plain text. It allows users to create formatted documents using simple syntax, which can be converted into HTML and other formats.

Why use Markdown?

  • Simplicity: Easy to write and read.
  • Portability: Works across different platform and tools.
  • Compatibility: Can be converted to various formats like HTML, PDF, etc.
  • Widely used: Commonly used in documentation, README files, blogs, and note-taking apps.
Basic Syntax

Headings

Use  #  followed by a space for headings. The number of  #  symbols determines the heading level. 

# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

Paragraphs & Line Breaks

Write text normally for paragraphs. To create a new paragraph, leave a blank line between two blocks of text.
To insert a line break within a paragraph, use two spaces at the end of a line.

This is a paragraph.
This is another paragraph.

This line has a  
manual line break.

Bold, Italics, and Strikethrough

  • Bold: Use double asterisks (**bold**) or double underscores (__bold__).
  • Italic: Use single asterisks (*italic*) or single underscores (_italic_).
  • Strikethrough: Use double tildes (~~strikethrough~~).
**Bold Text**
*Italic Text*
~~Strikethrough Text~~
Lists

Markdown supports both ordered (numbered) and unordered (bullet point) lists.

Ordered Lists

Unordered Lists

Use -, *, or + followed by a space to create an unordered list.

Indent items using two spaces or a tab to create sublists.

Links and Images

Markdown allows you to add hyperlinks and images using simple syntax.

Adding Links

Use square brackets [] for the text and parentheses () for the URL.

[Visit Google](https://www.google.com)

To add a tooltip when hovering over the link:

[Google](https://www.google.com "Go to Google")

You can also create reference-style links for better readability:

[OpenAI][1]

[1]: https://www.openai.com

Embedded Images

Use ![]() with an image URL or file path.

![Alt Text](https://example.com/image.jpg)

To add a tooltip (title) to the image:

![Alt Text](https://example.com/image.jpg "Image Title")

You can also use reference-style syntax for images:

![Logo][logo]

[logo]: https://example.com/logo.png
